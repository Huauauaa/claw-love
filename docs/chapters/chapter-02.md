# 第二章：并肩开发

Open Claw 第一次对外公测那天，林知夏的手机像被塞进了爆米花机。

社区通知一条接一条往外蹦：  
「机械臂抓取轨迹在低分辨率下抖动」  
「文档里环境变量拼写不一致」  
「Windows 下 demo 跑不起来」

她刚把第一条 issue 标成 `confirmed`，第二十条已经刷出来了。

“火了。”产品经理声音发飘，“我们真的火了。”

“先别兴奋，先分锅。”林知夏把椅子一转，冲周叙白扬下巴，“后端十条，前端十条，文档一起背。”

周叙白看了她一眼，没抬杠，只把工单面板按优先级重新排好：“高危我先处理。你把复现步骤统一模板，不然社区会互相抄错答案。”

“收到，周老师。”

“别阴阳。”

“这是尊称。”

两人嘴上照旧互呛，手上却配合得越来越像一套完整系统。  
林知夏负责把外部反馈翻译成能执行的任务，周叙白负责把任务拆成可落地的服务修改。一个拉近用户，一个兜底系统。

三天后，Open Claw 的 Star 数破万。公司临时决定：周末去参加城市开发者沙龙，做一场线下分享。

“你讲前端交互和开发者体验，我讲架构和稳定性。”周叙白看着议程表说。

“行。”林知夏点头，“别临场给我加‘顺便讲一下控制算法原理’这种地狱任务。”

“我没那么坏。”

“你最好是。”

---

周六下午，活动场地人满得像春运候车厅。  
林知夏在后台插上电脑，投影黑屏。

她换了 HDMI 线，还是黑。  
换接口，还是黑。  
主持人在台上已经报幕：“下面有请 Open Claw 团队——”

林知夏额头冒汗，低声骂了句：“完了，这比线上宕机还社死。”

周叙白从她手里接过遥控笔，声音很稳：“你把备用机开起来，我先上。”

“可我的部分——”

“我看过你全部稿子。”他看着她，“你写的每一页我都看过。先把场子接住。”

他转身上台，灯光落下来，周叙白拿着话筒，第一句就不是自己的开场白：

“大家好，我是 Open Claw 后端负责人周叙白。今天这场分享，本来最精彩的部分来自我的搭档林知夏。设备有点小意外，我先替她热个场，等她上来你们再听完整版。”

台下笑了，气氛一下松开。

林知夏抱着备用电脑站在侧台，听他把她写的每一个要点讲得一字不差——  
为什么把状态分成 `idle/running/retrying`，  
为什么要在 UI 层显式暴露错误级别，  
为什么“开发者体验”不是花哨皮肤，而是降低踩坑成本。

那些她熬夜改过十几版的句子，从他嘴里说出来，居然比她自己还顺。

十分钟后，投影恢复。林知夏接过话筒，先看了周叙白一眼，才笑着说：

“大家好，我是林知夏。刚才那段不算彩排事故，算我们团队做了个实时容灾演示。”

台下爆发一阵掌声。

分享结束后，一群开发者围上来提问，林知夏连喝口水的时间都没有。等最后一位合影结束，场馆已经开始收椅子。

她和周叙白并肩走出会场，夜风有点凉。

“你什么时候背下我稿子的？”她问。

“你改第三版的时候。”周叙白把外套递给她，“你在会议室自言自语练过三遍，声音太大，我不想听都不行。”

“偷听还这么理直气壮？”

“不是偷听，是风险预案。”

林知夏笑出声：“你这个人，连安慰都要写成事故复盘。”

周叙白没接话，只是把步子放慢了半拍，配合她的节奏。

---

回程地铁上，人不算多。林知夏抱着电脑包坐在靠窗位置，刚说完一句“我今天真是差点心梗”，下一秒就困得睁不开眼。

她头一点一点，最后轻轻靠在周叙白肩上。

周叙白僵了半秒，没动。  
过了会儿，他把手机亮度调到最低，单手回社区评论：

「Issue #287 已复现，修复将于明日合并。感谢反馈。」  
「文档 typo 已更正，欢迎继续提 PR。」

车窗映出两个人的影子，一个睡着，一个清醒；一个呼吸均匀，一个连翻页都尽量放轻。

林知夏在半梦半醒间听见地铁报站，没睁眼，只含糊问了一句：“到哪了……”

周叙白低声回她：“还没到。你睡吧。”

那一刻她忽然很确定，自己对这个“人形代码规范”的情绪，已经不只是“合作愉快”那么简单。
