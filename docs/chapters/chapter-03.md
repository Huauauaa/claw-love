# 第三章：误会与分岔

Open Claw 公测满一个月那天，公司把大会议室的投影换成了深蓝色模板，标题写得很像一份温和的判决书：

**《Open Claw 商业化推进方案》**

林知夏坐在第三排，看到第一页就皱起了眉。

“为保障商业竞争力，”负责人站在台上翻页，“从下个版本开始，核心控制策略延迟开源三个月；企业控制台、远程运维和权限审计将转入闭源模块；社区版只保留基础能力。”

会议室里安静了两秒。  
然后有人小声说：“这不就是先把人拉进来，再把门关上吗？”

林知夏没忍住，直接举手：“那我们之前公开承诺的路线图呢？上个月沙龙上还说会持续开放核心能力。”

负责人推了推眼镜：“承诺没有变，只是节奏调整。商业化是公司战略。”

“节奏调整”和“先不开源”，通常只差一个不愿直说的句号。

林知夏侧过头，下意识去看周叙白。

周叙白坐在靠窗的位置，神情很淡，手里转着一支黑色签字笔，从头到尾都没说话。

那一刻她心里忽然空了一下。

---

散会后，群里炸得像新开了十个讨论串。

产品经理：「知夏你先别上论坛，社区已经有人在问路线图是不是要改。」  
测试同学：「企业版功能清单谁来定？我现在连边界都不知道。」  
运维同学：「为什么安全审计那部分 suddenly 被拉进保密群？」

林知夏正盯着最后一句，手机又震了一下。

走廊尽头，磨砂玻璃会议室的门半掩着，门牌上贴着一张临时打印的纸：  
**Project Falcon - Restricted**

她本来只是路过，却在门缝里看见了熟悉的侧影。

周叙白坐在里面，旁边是 CTO、法务和新来的商业化负责人。投影上有一行字一闪而过：

`Open Claw Enterprise Auth / Security / License Gateway`

林知夏脚步一顿。

她没听清会议内容，只看见周叙白低头翻资料，胸前挂着一张新发的灰色权限卡。

保密项目组。  
企业认证。  
License Gateway。

几个词像没有注释的异常栈，瞬间把她脑子打满了红色警告。

晚上回到工位，她打开内部代码平台，原本共享的 roadmap 分支下面，多了一个她没有权限访问的仓库：

`openclaw-falcon`

最近提交人第一行就是——**zhouxubai**。

林知夏盯着屏幕，觉得胸口像被什么东西轻轻顶了一下，不疼，却闷得发酸。

她想起地铁上那句“你睡吧”，想起沙龙后台那句“我看过你全部稿子”，想起他替她接住场子的那十分钟。

原来那些靠近，也可以在一份保密协议前面安静后退。

---

接下来的一周，他们几乎没有单独说过话。

也不能算完全没说。  
只是所有交流都只剩下工单系统里冷冰冰的动词：

「请补充复现步骤」  
「已处理」  
「待验证」  
「驳回，风险过高」

林知夏把自己的工位搬去了靠社区运营更近的那一排，说是“方便处理反馈”。  
同事们表面不问，背地里却在茶水间交换眼神，像发现两个一直低延迟通信的服务，突然开始丢包。

周五晚上，社区论坛果然炸了。

有开发者发了长帖，标题写得很不客气：

**《Open Claw 是不是准备把开源项目做成企业版引流壳？》**

帖子下面跟了几百条评论：

「上个版本还说透明协作，这个版本就开始神秘仓库？」  
「如果核心控制逻辑延期开源，那我们贡献 patch 的意义是什么？」  
「是不是以后社区只是测试服？」

林知夏看得眼皮直跳，刚准备起草回应，商业化负责人就在群里发通知：

「所有对外口径统一，由市场部整理后发布。研发不要擅自回复。」

她盯着那句“不要擅自回复”，气得差点把回车键按出火星。

就在这时，周叙白在群里发了一句：

「收到。」

林知夏把聊天窗口关了。

动作快得像关掉一个再也不想看的弹窗。

---

真正把他们推向争执的，是发布前两天的那个深夜。

林知夏还在改文档页的警告提示，社区邮箱突然跳进来一封安全报告：

**存在权限绕过风险。**

报告人是一位海外开发者，复现步骤写得非常克制而专业：  
如果用户在旧版本控制台保留调试开关，再配合新接口的权限缓存缺陷，就有概率越权读取企业版设备列表。

邮件末尾还附了一句：

**I hope you disclose this responsibly before someone else does.**

林知夏一下清醒了。

这不是文案争议，这是实打实的安全漏洞。

她立刻把邮件转到应急群，艾特周叙白。十分钟后，周叙白把复现结论发了出来：

「问题成立。与 Falcon 网关改造有关，我来接。」

Falcon。

又是 Falcon。

林知夏直接起身，穿过半层办公区，推开了后端会议室的门。

周叙白正在白板前和两名同事对接口缓存策略，抬头看见她，手里的笔顿了一下。

“出来聊。”林知夏说。

走廊的感应灯亮起来，白得发冷。

“Falcon 到底是什么？”她压着声音问，“你进保密组，是为了把 Open Claw 的东西一点点关起来？”

周叙白看着她，沉默了半秒：“现在有更紧急的事，先修漏洞。”

“我问的不是优先级，我问的是你。”她声音有点发颤，“你是不是从一开始就认同公司这套做法？”

“不是你想的那样。”

“那你解释。”

周叙白指节收紧，像在克制什么：“有保密限制，我现在不能说。”

这句话像一把没有温度的刀，利落得过分。

林知夏忽然笑了一下，笑意却一点都没到眼底。

“不能说？”她看着他，“我们一起熬夜救项目、一起扛社区、一起在台上讲‘开发者体验不是噱头’。结果到了要站队的时候，你连一句实话都不能给我？”

“知夏——”

“你知道社区现在怎么骂吗？他们说我们拿开源做招牌，说得最响的人只是为了把用户带进闭源漏斗。”她盯着他，眼眶有点红，“我本来不信。可你站在保密会议室里，一句话都不说，我还能怎么信？”

周叙白也冷了下来：“你现在需要的是答案，还是一个能让你立刻判我有罪的证据？”

“至少我没有瞒着你。”

“你觉得我想瞒你？”

“那你就别总是什么都一个人扛着，扛到最后只剩别人像傻子一样被蒙在外面！”

最后一句落下时，整条走廊安静得只剩空调风声。

周叙白看着她，目光像被什么东西压住了，低低回了一句：

“如果我现在告诉你，只会把你一起拖进来。”

“可你现在这样，”林知夏喉咙发紧，“已经把我推开了。”

她说完转身就走，没再回头。

会议室门在身后合上，声音很轻，却像一次明确的断开连接。

---

第二天，漏洞消息还是泄露了。

一位安全研究员在社交平台发帖，虽然隐去了关键细节，但“Open Claw”“权限绕过”“企业网关”几个词已经足够把舆论点燃。

GitHub issue 区瞬间多出上百条新反馈：

「所以你们真的做了没公开的企业网关？」  
「请立即说明影响范围。」  
「闭源模块把开源项目安全边界一起搞坏了，是吗？」

市场部临时起草的回应模糊得像没写完的接口文档，发出去不到二十分钟就被喷上了首页。

公司高层连夜开会，意见分成两派：

一派主张先删帖、先压热度、等发布后再统一解释；  
另一派主张直接暂停版本，公开漏洞和商业化调整。

林知夏坐在边上，一句话没说。

她第一次发现，最让人疲惫的不是线上事故，而是有人想把事故伪装成“可控波动”。

凌晨一点，周叙白终于开口。

“我反对压消息。”他说。

会议室所有人都看向他。

他把电脑转过去，屏幕上是他刚整理好的影响分析和修复方案。

“Falcon 不是单纯的商业化项目。”周叙白声音很稳，“它最初是针对大客户场景做权限隔离和安全审计，目的是把企业能力从社区主干里拆出去，避免继续让两套需求互相污染。这个漏洞，恰恰证明现在的拆法不彻底。”

商业化负责人皱眉：“现在不是解释架构理想的时候，是控制风险的时候。”

“控制风险最有效的办法就是说实话。”周叙白看着他，“立刻冻结发布，公开漏洞编号、影响范围和修复时间；把网关相关改造从主仓库回退出去，恢复社区版独立边界；企业能力单独维护，接口文档公开，安全补丁同步披露。”

法务提醒：“这样会影响签约进度。”

“但能保住社区信任。”周叙白说，“Open Claw 走到今天，不是因为我们藏得好，是因为大家相信我们不会拿他们当试验田。”

林知夏抬起头，第一次正面看向他。

那一瞬间，她忽然意识到，自己好像漏读了一整段上下文。

---

天快亮时，会议才散。

周叙白最后一个从会议室出来，眼底有很重的倦色，像整夜没停机的服务器。

林知夏站在走廊尽头等他，手里攥着那杯早就凉透的咖啡。

“所以你进 Falcon，”她开口，声音轻了很多，“不是为了把项目卖掉？”

周叙白停下脚步。

“一开始不是。”他说，“后面有没有人想往那个方向推，有。但我进组，是因为上线前的权限模型一直有隐患，CTO 让我做隔离方案。我签了保密，不只是对外，也包括对项目组外的人。”

他顿了顿，又补了一句：

“包括你。”

这句解释并没有让误会瞬间消失，反而让林知夏心里更乱。

因为她忽然不知道，自己该怪他没有信任她，还是怪自己根本没等他把话说完。

“那你为什么连一点都不告诉我？”她问。

周叙白看着她，嗓音发哑：“因为你太在意社区，也太容易把责任往自己身上揽。如果我提前告诉你，你一定会冲出去和所有人对着干。到时候被夹在中间的人，不会只有我。”

林知夏低头盯着纸杯边缘，半天才说：“可你替我做决定的时候，也没问过我愿不愿意。”

周叙白没反驳。

这是他们第一次，都清楚地看见了对方的好意里藏着怎样的锋利边角。

“漏洞修复我已经提 PR 了。”过了一会儿，周叙白说，“早上九点前会合并。商业化方案也会重写。只是——”

“只是什么？”

“你如果还生气，可以继续不理我。”他看着她，眼里难得有一点近乎无措的疲惫，“但项目这次，别跟我分开站。”

林知夏鼻尖一酸，偏过头，没让他看见。

“我是在气你。”她低声说，“又不是在气 Open Claw。”

周叙白像是想笑一下，却只来得及把那点情绪压回去：“那就好。”

“你也别高兴太早。”林知夏抬眼看他，“信任这东西不是 hotfix，不能一键恢复。”

“我知道。”

“而且我还没原谅你。”

“嗯。”

“……但 PR 我会帮你看。”

天边已经泛白，办公室窗户映出两个人都没睡好的脸。

他们站得不算近，中间隔着一场误会、一份保密协议、一个差点失控的版本发布；  
可也没有远到彻底失去彼此。

周叙白把电脑递给她，页面停在那份待审的补丁上。

PR 标题写着：

**`fix: decouple enterprise gateway from community auth flow`**

林知夏接过电脑，指尖碰到他时，还是会下意识顿一下。

她没有躲开。

只是很轻地说：“先修版本吧。”

像在说项目。  
又不完全像只是在说项目。
